第一次解密

  今天学长给了个十分基础的题，就是破解一个压缩包密码，压缩包密码是六位纯数字，
直接暴力破解就行了。
  先下载压缩包，之后使用ARCHPR，设置条件，运行。  
  得到密码是444555。

第一篇博客——github+hexo博客的搭建过程

一、搭建环境

  1.安装node.js
    百度搜索node.js官网，下载node.js。
  2.注册github
    百度搜索github官网，注册github账号。建立仓库，仓库名称必须为xxxx.github.io。
    在仓库中新建一个index.html文件。输入<h1>Hello GitHub Pages</h1>。
    运行，打开之前注册的xxxx.github.io。可以看到刚才输入的hello github pages。
  3.下载git for windows
    下载完成后运行，输入git命令

二、hexo下载

    百度搜索hexo，选择 文档|hexo。点击文档，会出现安装密令。
    新建一个文件夹blog，cmd打开命令行，找到之前的文件夹，输入安装密令自动安装。
    输入hexo init 博客名称
    cmd到博客 输入 npm install安装
    安装完成后输入 hexo s -p 5555
    网址输入 local:5555

三、githubSHH配置

   1.在文件中找到git-bash.exe运行。
     输入密令 ssh-keygen -t rsa -C "自己的邮箱"。三次回车。
   2.登录github 找到设置 找到SSH and GPG keys 点击 New SSH key
     title随便写，在文件中找到id_rsa.pub，记事本打开。复制内容，粘贴到Key中。

四、sublime安装

   百度搜索 sublime下载
   打开sublime_text.exe，将博客目录拖进去。

五、hexo发布至GitHub

    打开sublime，找到左侧_config.yml。
    找到author：改为自己。
    找到language：改为zh-CN。
    找到url：改为https://你的网址。
    登录GitHub，找到仓库，找到Clone or download，选择http，复制。
    将_config.yml拉到最下面，找到type，输入git。
    在type下面一行输入repo:，将刚才复制的粘贴上。
    再下一行输入branch：master。
    cmd找到博客，输入hexo g，在输入hexo d。
    之后输入git config --global user.mail"你的邮箱"
    输入git config --global user.name"你的名字"
    再次hexo d，提示输入github账号密码。
    之后登录自己的网址就成功了！！

第一次写博客，图片还不会插入，许多地方表述不清，继续加油吧。
